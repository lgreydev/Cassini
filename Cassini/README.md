# Cassini

The application fetches data from the network, processes the URL outside of the ​main​ ​queue​! Navigation with **Split View**, working on iPhone and iPad. Use the **Scroll View** to display the images.

## Functionality used
 - Networking
 - Multithreaded
 - Split View
 - Activity Indicator
 - Scroll View
 - Image View
 - View Zooming
